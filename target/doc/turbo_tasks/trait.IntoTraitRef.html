Expand description

A trait that allows a value trait vc to be converted into a trait reference.

The signature is similar to IntoFuture, but we don’t want trait vcs to have the same future-like semantics as value vcs when it comes to producing refs. This behavior is rarely needed, so in most cases, .awaiting a trait vc is a mistake.
